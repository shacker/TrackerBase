TrackerBase Readme

Thankyou for installing TrackerBase, a toolkit for building BeOS web site extraction scripts. The following tools have been installed on your system:

  • The text/trackerbase filetype with a collection of custom attributes.

  • Filesystem indexes for several of these attributes.

  • Sample databases: Tips, News, and Simple.

  • Sample scripts: BePeople, Tips, and News.

  • A tutorial for those new to bash and shell scripting.

  • A code library containing the most common building blocks of TrackerBase scripts.

  • TrackerBase II - Dynamic (on the fly) web site generation examples for advanced users.

  • This documentation.

Sample scripts have been installed to /boot/home/config/bin and are ready to run -- just type a script's name into Terminal: 

TrackerBase.news
TrackerBase.tips
TrackerBase.people
In addition, a five-part tutorial is available. To run the tutorial, open the tutorial folder and move these files to /boot/home/config/bin: 
TrackerBase.simple.1a
TrackerBase.simple.1b
TrackerBase.simple.1c
TrackerBase.simple.1d
TrackerBase.simple.1e
