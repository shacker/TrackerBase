BeOS People Files --> HTML

The bepeople script runs through your /boot/home/people directory and creates an HTML index listing all of your personal and business contacts on a single page.

If you don't have any People files on your system, you'll find a file called people.tbase.zip in the bepeople projects folder. This contains 9 dummy files you can use for testing. Delete it if you already have Person files.

After running the TrackerBase.bepeople script, you can email anyone with a single click, or click on people's names to launch a separate details page. These pages won't win any design awards, but you can take care of that; I just wanted to get it working.

This script is the slowest of the lot, perhaps because it has to look up every single attribute. This is also the only script here to use the "read" command to get user input rather feeding the read command a text file. Enter your name at the prompt and it'll show up in the HTML docs. BePeople is the only script in the collection that doesn't use TrackerBase files -- it just applies the same principles to People files and their attributes.

The other interesting thing this script does is to use a toggle variable -- with every loop through, it flips a switch, which is what enables the table on the index page to be striped. The cell color variables are easily accessible -- just insert standard Netscape or hexadecimal colors (RoColor is a great tool for this!)

This script doesn't use queries, but it easily could. Excercise for the reader: hack this script to only display People who live in, say, Illinois and Alabama. Or re-write / re-arrange it to group people by the first letter of their last names, inserting a divider of a third color for each new letter.
